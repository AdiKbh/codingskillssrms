# Student Record Management System

Student Record Management system that allows the user to add, delete or edit existing data. This program also contains file handling to record the student details that have already been entered.

## Tech Stack

**Frontend**: React, Tailwind, CSS

**Backend**: Node JS, C programming for data handling
